<h2>Introduction</h2>
This project is a simple yet functional demonstration of a login and register interface built with ReactJS, focusing on utilizing React hooks for state management.

<h2>Installation</h2>
To run this project locally, follow these steps:
<br>
Clone the repository: git clone https://github.com/lauradsc/form-and-login-with-ReactJS.git<br>
Navigate to the project directory: cd my-app <br>
Install dependencies: npm install <br>

<h2>Usage</h2>
After installation, you can start the development server: <br>

npm start <br>
Access the application in your web browser at http://localhost:3000.

<h2>Technologies Used</h2>
ReactJS: Frontend library for building user interfaces.<br>
React Hooks: Utilized for managing component state.<br>
